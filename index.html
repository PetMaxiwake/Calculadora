Modo ERP: ao carregar um produto, o Custo de Compra e o Custo Médio vêm direto do banco (já calculados pelo sistema — sem estimativa de %). O Preço de Venda vem preenchido mas é editável para simular. O Preço Ideal p/ Compra, nesse modo, não desconta créditos de ICMS/IPI/PIS/Cofins na compra (não temos essa quebra por produto nas duas tabelas usadas).
Modo manual (sem produto carregado): todos os campos com fundo amarelo são livres para simular, e o Custo de Compra é calculado a partir das % informadas.
* Pr. Mín. Ct. Médio usa o Custo Médio do ERP quando um produto está carregado; em modo manual, replica o Preço Mínimo (não há custo médio separado disponível).
A busca de produtos só funciona quando a página está hospedada em http/https (ex: GitHub Pages) — abrir o arquivo direto (duplo clique) não carrega a base de dados.
